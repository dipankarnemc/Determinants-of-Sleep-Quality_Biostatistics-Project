# Data Description

The dataset used in this project is part of the  
**CitieSHealth Barcelona Panel Study (2020–2021)**.

Because the raw dataset cannot be redistributed publicly,  
A placeholder file is included instead.  

Variables used in this analysis:
- sleep_quality (0–10)
- stress (0–10)
- alcohol (Yes/No)
- pm25bcn (µg/m³)
- date_all
- ID_Zenodo
